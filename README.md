# ts-library-template
Typescript library template

## Step to start !
- npm install
- node scrtips/initConfig.js

## Lint
```
npm run lint
```

## Unit Test
```
npm run test
```

## Build
```
npm run build
```

## Examples
```
es : npm start 
cjs : node index.js
umd : npx serve
```
